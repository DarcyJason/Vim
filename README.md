# 安装方法

## 请确保安装curl

### For macOS

```bash
brew install curl
```

### For Debian

```bash
apt install curl
```

## 输入以下内容自动安装(默认是Bash)

```bash
cd ~ && /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)" && brew install git && git clone https://github.com/DarcyJason/Vim.git && bash ~/Vim/install.sh
```
